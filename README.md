## Getting Started

### Installing

Clone

```sh
git clone
```

change directory

```sh
cd gruppe-70
```

install dependencies

```sh
npm install
```

### Running

#### Build - production

Build project,

```sh
npm run build
```

Then you should be good to go.

```sh
npm start
```

A dev server will open locally on port 3000. The content on this server can be accessed in any browser by navigating to the URL:

```sh
http://localhost:3000
```

#### Development

Run development script

```sh
npm run dev
```
